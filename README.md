# 🏨 Hospitality Data Analysis Dashboard
## 📌 About the Project

This project focuses on analyzing hospitality industry data to understand revenue trends, booking patterns, and customer behavior. The dashboard is built using Power BI and provides meaningful insights to support business decision-making.

## 🎯 Objectives
- Analyze hotel booking trends
- Understand customer behavior
- Identify revenue patterns
- Compare weekday vs weekend performance
- Reduce cancellations and improve occupancy
## 🛠 Tools & Technologies
- Power BI (Data Visualization)
- Microsoft Excel (Data Processing)

## 📈 Key Insights
- Elite room class generates the highest revenue.
- Majority of bookings are successfully completed.
- Weekend bookings are higher than weekdays.
- Revenue varies significantly across cities and properties.
- Weekly trends show fluctuations in bookings and revenue.
- High cancellation rate (~25%) indicates improvement opportunity.
